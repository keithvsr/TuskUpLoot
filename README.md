## TuskUpLoot (TBC Classic Anniversary)

Minimal World of Warcraft addon for **TBC Classic Anniversary** (`Interface 20505`) with a toggleable in-game window showing basic character info and stats.

### Install

- Copy the `TuskUpLoot/` folder into your WoW Anniversary client AddOns directory.
- Relaunch the game or run `/reload`.
- Ensure the addon is enabled on the character select screen.

### Usage

- **Toggle window**: `/tul` (alias: `/tuskup`)
- Drag the window by its title area.
- Close button hides the window.

### Build a distributable folder

Run:

```bash
./build.sh
```

Output will be created at `dist/TuskUpLoot/` and can be copied directly into your AddOns directory.
